# 과제 - 02. 일상 작업

## 문제 1

아래 시나리오를 재현하는 스크립트를 작성하세요.

1. 임시 저장소를 초기화한다.
2. `notes.txt` 에 세 줄("할 일 1", "할 일 2", "할 일 3")을 작성하고 커밋한다.
3. "할 일 4"를 추가한 뒤 `git add notes.txt`를 실행한다.
4. "할 일 5"를 추가한다 (스테이징하지 않음).
5. `git diff` 와 `git diff --staged` 를 각각 출력해 두 결과가 다름을 확인한다.

- 파일명: `hw01.sh`

```bash
# 예상 출력 (핵심 부분)
=== diff (워킹 vs 스테이징) ===
+할 일 5

=== diff --staged (스테이징 vs HEAD) ===
+할 일 4
```

## 문제 2

다음 조건을 모두 만족하는 스크립트를 작성하세요.

1. 임시 저장소를 초기화하고 커밋을 5개 만든다.
2. `.gitignore` 에 `*.log` 패턴을 추가하고 커밋한다.
3. `debug.log` 파일을 생성했을 때 `git status` 에 나타나지 않음을 확인한다.
4. `git log --oneline` 으로 전체 커밋 목록을 출력한다.
5. `git log --oneline -n 3` 으로 최근 3개만 출력한다.

- 파일명: `hw02.sh`
